# Starfield-Project

## Overview
A colorful, interactive rave-themed starfield particle system built with HTML5 Canvas. Features dynamic rainbow-colored 5-pointed stars that cycle through the HSL color spectrum with multiple effect modes and real-time controls.

## Structure
- `index.html` - Complete single-page application (HTML + CSS + JS)
- `spec.md` - Full project specification document

## Live URL
https://aiml-1870-2026.github.io/penguinprotector/Starfield-Project/

## Key Features
- Rainbow starfield with 5-pointed star shapes
- Effect modes: Kaleidoscope, Chromatic Aberration, Mouse Reactive, Rotation
- Real-time sliders: Star Count, Speed, Trail Length, Star Size, Spread, Bloom Intensity
- Motion blur trails and bloom/glow effects

## Tech Stack
- Pure vanilla JavaScript (no frameworks)
- HTML5 Canvas 2D rendering
- Class-based OOP (Star, BackgroundParticle)
- requestAnimationFrame animation loop
