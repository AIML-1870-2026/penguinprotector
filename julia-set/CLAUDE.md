# Julia Set Explorer

## Assignment Context
Interactive Julia Set fractal explorer built from `julia-set-explorer-spec.md`.

## Features
- Julia Set, Mandelbrot, and Burning Ship fractal rendering
- Collapsible controls panel with parameter sliders
- Preset Julia sets (Dendrite, Spiral, Douady Rabbit, San Marco, Dragon, Siegel Disk)
- Multiple color schemes (Classic, Fire, Ocean, Electric, Grayscale, Rainbow)
- Smooth coloring algorithm
- Zoom/pan with mouse and keyboard
- Parameter animation (Circle, Figure-8 paths)
- Split view mode (Mandelbrot + Julia)
- Image export (PNG)
- Educational orbit visualizer
- Keyboard shortcuts

## Tech
- Single HTML file with embedded CSS and JS
- HTML5 Canvas rendering
- No external dependencies
