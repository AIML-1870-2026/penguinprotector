♠ About AI Blackjack

This is a fully playable Blackjack game powered by the Anthropic Claude API. After each deal, an AI agent analyzes the game state and recommends the optimal action.

How it works: The agent receives your hand, the dealer's up card, and available actions. It returns a structured JSON object — not a prose response — so the recommended action is parsed precisely with no ambiguity.

Key loaded locally: Your .env file is read in-memory only. The key is never written to localStorage, sessionStorage, cookies, or any server. It is used solely for the direct API call to Anthropic.

Stretch features: The Basic Strategy Matrix highlights the current hand in real time. The Session Analytics panel tracks win rate, bankroll history, and how often the AI agrees with pure basic strategy.
