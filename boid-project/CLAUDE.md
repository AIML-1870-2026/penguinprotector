# Boids Rave Lab

## Project Overview
An interactive, rave-themed boids simulation that transforms flocking behavior into a vibrant visual experience.

## Files
- `index.html` - Main HTML structure with canvas and UI controls
- `styles.css` - Rave theme styling with glassmorphism and neon effects
- `script.js` - Boids physics engine, rendering, and event handlers

## Core Features
- 5 parameter sliders (separation, alignment, cohesion, neighbor radius, max speed)
- 3 behavior presets (Rave Swarm, Chaos Mosh, Bass Drop Cluster)
- Real-time instrumentation (FPS, boid count, avg speed, avg neighbors)
- Pause/Resume, Reset, and Boundary mode controls
- Neon rave aesthetic with glow effects and motion trails
